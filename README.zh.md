# Notion Flow — Obsidian 插件

[English](README.md)

为 Obsidian(桌面端)带来 Notion 风格的编辑体验:块拖拽、斜杠命令、浮动格式工具栏,以及更简洁的实时预览渲染。界面支持简体中文和英文,自动跟随 Obsidian 的语言设置。

## 安装

### 手动安装

1. 从 [最新 Release](../../releases) 下载 `main.js`、`manifest.json`、`styles.css`。
2. 打开你的库文件夹,进入 `.obsidian/plugins/`(没有 `plugins` 就新建一个;`.obsidian` 是隐藏目录,Finder 中按 `Cmd+Shift+.` 显示)。
3. 新建文件夹 `notion-flow`,把三个文件复制进去。
4. 在 Obsidian 中:设置 → 第三方插件 → 如有提示先关闭安全模式 → 点刷新图标 → 启用 **Notion Flow**。
5. 确保使用**实时预览**模式编辑(设置 → 编辑器 → 默认编辑模式)。

## 功能

**块拖拽** — 悬停任意一行,左侧出现 ⋮⋮ 拖拽手柄,整个块会高亮显示,拖起后有浮动预览跟随光标;带圆点的指示线标出落点,鼠标的**横向位置决定嵌套层级**——按你的库缩进设置(制表符或空格,与 Tab 键一致)逐级步进,并自动吸附到合法的 Markdown 层级,块会自动重新缩进。只横向拖动(不上下移动)可以原地改变缩进层级。列表项会带着子项一起移动,标注/引用和代码块作为整体移动(包括列表里的代码块),拖到窗口上下边缘会持续自动滚动,按 `Esc` 可取消拖拽。

**块菜单** — *点击*(不拖动)⋮⋮ 手柄:转换为(正文、标题、列表、待办、引用)、创建副本、复制文本、删除块。

**键盘操作** — `Alt+↑`/`Alt+↓` 上下移动光标所在的块(代码块整体移动);`Alt+Shift+D` 复制块。可在快捷键设置中改绑。

**斜杠命令** — 在行首、空格后、或中文文字后直接输入 `/`(无需空格),继续输入即可过滤,中英文关键词都支持(`/h1`、`/表格`):标题、无序/有序/待办列表、引用、标注、折叠块、代码块、表格、分割线、图片嵌入、内部链接。在代码块内不会误触发;`/分割线` 会自动与上一行保持空行,避免上一行被解析成 Setext 标题。

**浮动工具栏** — 用鼠标或键盘(Shift+方向键、`Cmd/Ctrl+A`)选中文本即弹出:加粗、斜体、删除线、**文字颜色**(8 色)、**高亮颜色**(默认 `==` 加 8 种色调)、行内代码、链接、**清除格式**。已应用的格式按钮会点亮;对已着色的文字选新颜色会原地换色,⊘ 按钮移除颜色。`Esc` 关闭工具栏。颜色以内联 HTML 写入(`<span style="color:…">` / `<mark …>`),使用 Obsidian 主题的调色板变量,自动适配明暗主题。

已知限制:完全在渲染组件*内部*选中的文本(如标注预览里、双击已着色的词),是 Obsidian 无法映射回编辑器的 DOM 选区,工具栏无法处理——把选区扩到组件边界之外,或先用 `</>` 按钮展开标注源码。

**粘贴 URL 生成链接** — 选中文本后粘贴 URL,自动变成 `[文本](链接)`,和 Notion 一样。

**嵌套块视觉缩进** — 列表项里的代码块和标注会连同背景一起右移,像 Notion 一样呆在父项下方,而不是贴着左边缘。

**更简洁的渲染** — 实时预览中:弱化列表标记、隐藏 `>` 引用标记、柔化分割线、圆角强调色复选框、完成的待办淡出、行内代码胶囊样式、编辑行上的语法符号(`**`、`#` 等)几乎透明。缩进参考线使用 Obsidian 原生实现,插件不重复绘制。

所有功能都可以在 设置 → Notion Flow 中单独开关。

## 语言

插件跟随 Obsidian 的界面语言(设置 → 通用 → 语言),内置简体中文和英文。斜杠命令搜索在任何界面语言下都同时支持中英文关键词。欢迎提交 PR 增加更多语言——见 [`src/i18n.ts`](src/i18n.ts)。

## 开发

```bash
npm install --legacy-peer-deps   # obsidian 的 peer 依赖固定了旧版 @codemirror/state
npm run typecheck                # tsc -noEmit
npm test                         # 用 stub 打包后运行 test/run*.mjs
npm run build                    # 生成 main.js
```

把 `main.js`、`manifest.json`、`styles.css` 复制到 `<库>/.obsidian/plugins/notion-flow/`,重载插件(或重启应用)即可在 Obsidian 中测试。
