# Notion Flow — 5 分钟上手

[English version](notion-flow-demo.md)

> 将本文档复制到 Obsidian 库中，再创建一份练习副本，并在桌面端使用**实时预览**打开。下文用到的大多数功能已在**设置 → Notion Flow** 中默认开启。

## 1. 移动和创建内容块

悬停本段。拖动 `⋮⋮` 可移动整个段落，点击 `⋮⋮` 可打开块菜单，点击 `+` 会在下方创建内容块并打开斜杠菜单。

拖动时左右移动指针，可以改变嵌套层级。试着移动下面的父列表项，其中的嵌套内容会一起移动。

- 一个项目进展
  - 初稿已经完成
  - 评审已经排期

  ```ts
  const project = { status: "ready" };
  project.status;
  ```

  > 嵌套引用会与父列表项保持对齐。

  > [!tip] 嵌套 Callout
  > 标题和正文会作为一个完整内容块移动。

单击手柄而不拖动，试用**转换为**、**创建副本**和**复制文本**。只对不介意移除的内容使用**删除块**。

## 2. 使用斜杠命令插入内容

把下面的练习文字替换成 `/`，然后继续输入以筛选菜单。中英文关键词都可以使用。

练习行——在这里输入 `/`。

常用搜索包括 `/h1`、`/h2`、`/h3`、`/bullet`、`/number`、`/todo`、`/quote`、`/callout`、`/toggle`、`/code`、`/table`、`/divider`、`/image` 和 `/link`。

如需自定义表格尺寸，请用鼠标选择**表格**。在尺寸选择器中拖过网格，或使用方向键调整后按 `Enter`。如果直接在斜杠结果上按 `Enter`，会插入 3 × 3 表格。

## 3. 编辑表格

点击下面已渲染表格中的单元格，打开表格工具栏。

| 任务 | 负责人 | 状态 |
| :--- | :--- | :---: |
| 整理文档大纲 | 小林 | 已就绪 |
| 检查示例 | 小梅 | 进行中 |
| 发布指南 | 小周 | 未开始 |

试试这些操作：

- 新增一行或一列，然后再删除。
- 设置当前列的对齐方式。
- 设置单元格底色和表格底色，再移除其中一种颜色。
- 选择**格式化表格**。
- 按 `Alt+F10`，再用左右方向键在工具栏按钮之间移动。
- 点击表格的 `⋮⋮` 手柄，打开整表操作或移动完整表格。

### 原始 Markdown 表格练习

下面的导航只作用于**源码模式**中的表格，或仍未渲染的原始 `|` 表格。

1. 在空行中输入 `| 名称 | 数量`，按 `Tab` 自动补全表格结构。
2. 使用 `Tab` 和 `Shift+Tab` 在单元格之间移动。
3. 使用 `Enter` 移动到当前列的下一行。
4. 从最后一个单元格继续导航，自动追加一行。
5. 在空的最后一行按 `Enter`，移除该行并离开表格。

## 4. 设置文本格式

选中下一句话的一部分。使用浮动工具栏设置粗体、斜体、下划线、删除线、文字颜色、高亮、行内代码或链接，最后再试试**清除格式**。

选中并格式化这句话，全程不离开实时预览。

标准 Markdown 示例：**粗体**、*斜体*、~~删除线~~、`行内代码`、==高亮==和[外部链接](https://obsidian.md/)。

如需尝试粘贴 URL，请选中文字“Obsidian 帮助”，然后粘贴 `https://help.obsidian.md/`。启用**粘贴 URL 生成链接**后，选中文字会变成 Markdown 链接。

启用**隐藏 HTML 格式标签**后，实时预览会隐藏下划线和颜色标签。**隐藏行内 Markdown 标记**默认关闭，可以开启后比较两种编辑方式。

## 6. 试用快捷键

将光标放到下面的练习段落中。

- `Alt/Option+↑` 或 `Alt/Option+↓` 移动内容块。
- `Alt/Option+Shift+D` 复制内容块。
- 可以在 Obsidian 的**快捷键**设置中重新绑定这些块快捷键。

练习段落——移动或复制我。

## 7. 如果功能没有出现

- 块手柄需要桌面端实时预览，并启用**块拖拽**。
- 文本和就地表格工具栏需要启用**浮动格式工具栏**。
- 原始表格的 `Tab` 和 `Enter` 行为需要启用**表格编辑增强**。
- 完全位于已渲染 Callout 或彩色文字中的选区可能无法映射回编辑器。请打开其源码，或把选区扩展到普通编辑器文本。
- 如需重新开始，请使用**设置 → Notion Flow → 恢复默认设置**，然后用原始示例替换当前练习副本。
